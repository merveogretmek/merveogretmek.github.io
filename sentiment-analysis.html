Project Overview

This Bachelor's Thesis project titled "Machine Learning: A powerful tool for enhancing customer experience through sentiment analysis" explores how businesses can leverage sentiment analysis to interpret large volumes of customer reviews from Amazon. The research investigates how automating the classification of text into positive, negative, and neutral categories helps companies rapidly identify trends, strengths, and areas for product improvement using data-driven decisions.

Technical Stack

  • Programming Language: Python 3.7+
  • Data Processing: pandas, numpy
  • Machine Learning: scikit-learn
  • Lexicon Analysis: vaderSentiment
  • Dataset: Amazon Customer US Reviews Dataset (2015-2020)
  • Text Vectorization: TF-IDF, CountVectorizer

Key Research Findings

  • Logistic Regression Performance: Achieved highest accuracy (~93%) for two-class sentiment analysis
  • VADER Effectiveness: Lexicon-based approach achieved ~90% accuracy for binary classification
  • Category Bias Impact: Training on multi-category data yields more robust models across product types
  • Three-Class Challenge: Adding neutral class reduces overall accuracy, highlighting complexity of mixed sentiment identification
  • Cross-Category Analysis: Model performance varies significantly when testing across different product categories

Project Structure & Implementation

  • sentiment_models.py: Comprehensive comparison of 8 different ML algorithms
