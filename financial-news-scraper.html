Daily Bulletin Scrapers

Web Scraping • PDF Processing • Selenium Automation • Python

Project Overview

A sophisticated Python-based scraping system that automatically downloads, parses, and processes daily financial bulletins from major Turkish brokerage firms. The system extracts company news items from PDF documents, performs intelligent ticker symbol matching against BIST-traded stocks, and aggregates all data into a structured CSV format for comprehensive financial analysis and market research.

Technical Stack

  • Programming Language: Python 3.7+
  • Web Automation: Selenium WebDriver with ChromeDriver
  • PDF Processing: pdfminer for text extraction
  • Data Processing: pandas, numpy
  • Web Requests: requests library
  • Fuzzy Matching: fuzzywuzzy for ticker symbol validation
  • Data Storage: CSV file format for structured data
  • Text Processing: Regular expressions for pattern matching

Brokerage Firms Covered

  • Oyak Yatırım: Direct PDF download and parsing
  • Piramit Menkul Kıymetler: Selenium-based navigation and extraction
  • Tacirler Yatırım: Automated web browsing with Selenium
  • Vakıf Yatırım: Dynamic content handling with Selenium
  • Ziraat Yatırım: PDF processing with direct download

Key Features & Workflow

  • Automated daily PDF bulletin downloads from multiple brokerage websites
  • PDF-to-text conversion using pdfminer for content extraction
  • Regular expression-based parsing to identify "company news" sections
  • Intelligent ticker symbol extraction and validation against BIST stock codes
  • Fuzzy matching algorithms to ensure accurate symbol identification
  • Structured data aggregation with timestamp and source tracking
  • Master CSV file management for consolidated financial data
  • Robust error handling for PDF format changes and website updates
  • Logging system for monitoring daily operations and data availability

Script Architecture

  • oyak_yatirim.py: Handles Oyak Yatırım bulletin downloads and parsing
  • piramit_yatirim.py: Selenium-based scraper for Piramit Menkul Kıymetler
  • tacirler_yatirim.py: Automated extraction from Tacirler Yatırım bulletins
  • vakif_yatirim.py: Dynamic content handling for Vakıf Yatırım
  • ziraat_yatirim.py: Direct download processing for Ziraat Yatırım
  • Shared CSV Database: Centralized storage for all extracted news items
  • BIST Ticker Reference: Master list for fuzzy matching validation

Technical Challenges & Solutions

  • PDF Format Variations: Implemented robust parsing with pdfminer to handle different bulletin layouts
  • Dynamic Web Content: Used Selenium WebDriver to handle JavaScript-heavy brokerage websites
  • Ticker Symbol Accuracy: Deployed fuzzywuzzy for intelligent matching against BIST reference data
  • Error Resilience: Built comprehensive exception handling for PDFSyntaxError and NoSuchElementException
  • Data Consistency: Standardized date/time formatting and source attribution across all scrapers
  • Website Changes: Modular design allows easy updates when brokerage sites modify their structure
