Projects

Here are some of the projects I've worked on. Each project showcases different aspects of my data science skills and expertise.

Sentiment Analysis on Amazon Customer Reviews

A comprehensive Bachelor's Thesis project exploring how machine learning enhances customer experience through sentiment analysis. Analyzes Amazon reviews (2015-2020) using multiple ML algorithms including Logistic Regression, Decision Trees, and VADER lexicon-based approaches, achieving 93% accuracy in sentiment classification.

Financial News Scraper

An automated web scraping system that collects and analyzes financial news from multiple sources. Built with Python and BeautifulSoup, the system extracts market insights, tracks sentiment trends, and provides real-time financial data aggregation for investment analysis and market research.

Triage Tool App

A smart healthcare triage application that uses machine learning algorithms to assess patient symptoms and prioritize medical care. Built with Python and React, the system provides real-time decision support for healthcare professionals, improving patient flow and emergency response efficiency in medical facilities.

Webform Filling Automation

An intelligent automation tool that streamlines repetitive webform filling tasks using Python and Selenium. The system features smart data mapping, dynamic form recognition, and batch processing capabilities, reducing manual data entry time by 85% while maintaining high accuracy and reliability across various web platforms.
